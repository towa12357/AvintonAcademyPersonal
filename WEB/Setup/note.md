## Procedure
- Install
```
sudo apt-get install curl
curl --silent --location https://deb.nodesource.com/setup_16.x | sudo bash -
sudo apt install nodejs npm
```
- Verify
```
node -v
sudo npm install npm -g
```
## Review
- npm was not installed at the same time as Node.js
